[Ansible](http://www.ansible.com) role to deploy [drone.io](http://readme.drone.io)

### Usage

```yml
drone_secret: 'my-secret'
drone_github_client_id: 'my-client-id'
drone_github_client_secret: 'my-client-secret'
```

### Defaults

```yml
drone_version: 0.5
drone_ip_addr: 0.0.0.0
drone_port: 80
```
